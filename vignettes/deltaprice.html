Delta Price Scrapper

Lex Liu

2018-12-12

deltaprice package aims to scrape Delta Airline ticket price. Given a list of departure and return dates, this package will help you collect ticket prices. This package aims to help people plan the most budget trip. Also, this package works as a starter code for anyone thinking to scrape a dynamic webpage using Selenium in R.

Dependencies

Before running RSelenium, the following is required:

  • install a version of java SE on your machine
  • download a standalone selenium server

When you have the items above ready, run the following command in the terminal prior to scraping. Remember to navigate to the folder where the .jar file sits and replace 3.9.1 with the version on your machine. Also, the selenium server above works for most of the mordern browsers, and if it is not the case for you, visit Selenium for more options.

    java -jar selenium-server-standalone-3.9.1.jar

Example

library(RSelenium)
library(deltaprice)
remDr <- remoteDriver(browserName="chrome", remoteServerAddr="localhost", port=4444L)
remDr$open()
